# Expense Tracker - Трекер расходов

## Автор
Белоусов Алексей

## Описание программы
Expense Tracker - это GUI-приложение для отслеживания личных расходов. Программа позволяет:
- Добавлять записи о расходах (сумма, категория, дата)
- Просматривать все расходы в удобной таблице
- Фильтровать расходы по категории и периоду
- Подсчитывать общую сумму расходов за выбранный период
- Сохранять и загружать данные в формате JSON

## Требования
- Python 3.6 или выше
- Библиотеки: tkinter (встроенная), tkcalendar
